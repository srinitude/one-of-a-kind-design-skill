# CREATIVE DIALS
> Generated from visual-styles-taxonomy v2.1.0 | one-of-a-kind-design skill

Four tunable parameters that control how a style is implemented. The same style at different dial settings produces radically different outputs -- this is how the taxonomy breaks free from style-to-use-case cliches.

---

## 1. Design Variance

**Range:** 1-10
**Description:** How experimental the layout and composition are

### Tier Definitions

| Tier | Range | Description |
|---|---|---|
| Conservative | 1-3 | Centered symmetric layouts, standard grid, safe hierarchy, corporate-appropriate |
| Creative | 4-6 | Asymmetric touches, mixed section rhythms, overlapping elements, editorial flair |
| Radical | 7-10 | Broken grid, masonry, extreme negative space, elements bleeding off-screen, art-direction-level composition |

### Per-Category Guidance

| Category | Starting Range | Rationale |
|---|---|---|
| Historical / Fine Art | 5-7 | These styles have inherent compositional opinions |
| Modern Digital / UI | 3-5 | Systematic styles need controlled variance |
| Illustration / Graphic | 5-8 | Illustration styles thrive with spatial freedom |
| Cultural / Regional | 6-9 | Cultural styles lose authenticity when over-gridded |
| Generative / Algorithmic | 7-10 | Generative styles ARE variance |
| Photography / Rendering | 4-6 | Let the image drive, layout serves |

## 2. Motion Intensity

**Range:** 1-10
**Description:** How much and how dramatically elements move

### Tier Definitions

| Tier | Range | Description |
|---|---|---|
| Static | 1-3 | Color shifts on hover, simple opacity fades, no scroll effects |
| Fluid | 4-6 | Spring-physics hover states, staggered scroll entries, smooth page transitions |
| Cinematic | 7-10 | Magnetic cursor, parallax stacking, scroll-driven choreography, ambient particle systems |

### Per-Category Guidance

| Category | Starting Range | Rationale |
|---|---|---|
| Historical / Fine Art | 1-5 | Reverence and restraint; surrealism can push to 7 |
| Modern Digital / UI | 3-7 | Motion is functional feedback, not spectacle |
| Illustration / Graphic | 2-5 | Let the illustration be the star, motion supports |
| Cultural / Regional | varies | Wabi-sabi at 1-3, psychedelic at 8-10 |
| Generative / Algorithmic | 6-10 | Motion IS the medium |
| Photography / Rendering | 3-6 | Cinematic pacing, not hyperactivity |

---

## 3. Visual Density

**Range:** 1-10
**Description:** How much content occupies a given viewport

### Tier Definitions

| Tier | Range | Description |
|---|---|---|
| Spacious | 1-3 | One element at a time, dramatic white space, luxury breathing room |
| Balanced | 4-6 | Standard app/site density, readable rhythm |
| Dense | 7-10 | Dashboards, data tables, multi-panel interfaces, minimal padding |

### Per-Category Guidance

| Category | Starting Range | Rationale |
|---|---|---|
| Historical / Fine Art | 2-5 | Art needs room |
| Modern Digital / UI | 4-8 | Functional density matched to purpose |
| Illustration / Graphic | 3-6 | Illustrations need framing |
| Cultural / Regional | 2-6 | Cultural styles often value emptiness (ma, negative space) |
| Generative / Algorithmic | varies | Noise fields sparse, cellular automata dense |
| Photography / Rendering | 2-5 | Photos need to breathe |

---

## 4. Audience Formality

**Range:** 1-10
**Description:** How formal or casual the overall tone reads

### Tier Definitions

| Tier | Range | Description |
|---|---|---|
| Casual | 1-3 | Playful, rounded, warm, conversational -- apps for consumers, creators, kids |
| Professional | 4-6 | Clean, trustworthy, competent -- SaaS, productivity, enterprise |
| Authoritative | 7-10 | Sharp, prestigious, editorial -- finance, law, luxury, institutional |

### Key Insight

This dial is independent of visual style. Art Deco at formality 3 is a retro cocktail bar app. Art Deco at formality 9 is a private bank. Same style, different implementation. This is how the taxonomy breaks free from style-to-use-case cliches.
