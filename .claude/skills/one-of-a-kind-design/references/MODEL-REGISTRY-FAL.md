## Tier Legend

| Tier | Description |
|------|-------------|
| **fast** | Ultra-low cost, rapid iteration, prototyping |
| **standard** | Balanced quality and cost for general use |
| **pro** | Professional-grade output for production |
| **premium** | Highest quality, flagship models for hero assets |
| **editing** | Specialized for image modification rather than generation |
| **utility** | Infrastructure/pipeline tool, not a generation model |

## Usage Pattern

All models are accessed via the `@fal-ai/client` SDK:

```typescript
import fal from "@fal-ai/client";

fal.config({ credentials: Bun.env.FAL_KEY });

const result = await fal.subscribe("fal-ai/flux-2-max", {
  input: {
    prompt: "...",
    image_size: "landscape_16_9",
  },
  logs: true,
  onQueueUpdate: (update) => {
    if (update.status === "IN_QUEUE") {
      console.log(`Queue position: ${update.queue_position}`);
    }
  },
});
```
